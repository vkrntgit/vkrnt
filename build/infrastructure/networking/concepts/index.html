Networking concepts


301 and 302 redirects

A 301 redirect is a status code that tells users and search engine that the page has moved permanently. A 302 redirect is used for temporary redirects. More here.

NAT'ing (NATting)

NAT stands for network address translation. This is the process where you use a firewall to assign a public IP to a computer inside a private network. Helps limit the number of public IP addresses an organization uses by allowing private IP networks to use unregistered IP addresses, but then converting to globally unique.
